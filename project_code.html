Project Code

This page documents the STM32 firmware used in the Smart Elevator System. The firmware was developed in STM32CubeIDE using C language and implements CAN communication between the elevator controllers.

STM32 CAN Controller

  • Defines CAN IDs for each controller node.
  • Sends floor request messages over the CAN bus.
  • Receives CAN messages from other controllers.
  • Processes Floor 1, Floor 2, and Floor 3 requests.
  • Displays test results using UART and Tera Term.

CAN IDs Used


#define SC_ID 0x100
#define EC_ID 0x101
#define CC_ID 0x200
#define F1_ID 0x201
#define F2_ID 0x202
#define F3_ID 0x203

Implementation Summary

The STM32 controller was successfully configured to communicate over the CAN network. UART output was used to verify the received messages and confirm that the elevator system responded to the requested floor commands.

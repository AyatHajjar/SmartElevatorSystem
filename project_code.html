STM32 CAN Controller Code

This page documents the STM32 firmware used in the Smart Elevator System. The code was written in C using STM32CubeIDE and is responsible for sending and receiving CAN messages between the elevator controllers.

CAN IDs


#define SC_ID 0x100
#define EC_ID 0x101
#define CC_ID 0x200
#define F1_ID 0x201
#define F2_ID 0x202
#define F3_ID 0x203

Code Description

The firmware defines unique CAN IDs for each elevator node. Each floor controller sends a CAN message when its button is pressed. The Supervisory Controller and Elevator Controller receive these messages and respond based on the requested floor.

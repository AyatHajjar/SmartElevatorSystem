Smart Elevator System

The Smart Elevator System is a Project VI website created to organize the main information about the elevator project. The website includes project information, a project plan, log book pages, login access, request access, and project documentation.

The purpose of this project is to build a simple web-based system that presents the project clearly and professionally. The team can use the website to show progress, responsibilities, design information, and important project documents.
